Visual Studio Code is an integrated development environment made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.


Download and install VS Code. Create a new file. See an overview of the user interface. Install support for your favorite programming language. Change your keyboard shortcuts and easily migrate from other editors using keybinding extensions. Customize your editor with themes. Then start coding!
